# Project Name

Joe and Brandon's Newsweek clone project

![screenshot](images/screenshot.png)

Additional description about the project and its features.

This project was the first project compiled by Brandon and Joe in the Microverse HTML/CSS section of the course work. It involved recreating a Newsweek webpage.

## Built With

- HTML and CSS
- VScode and Atom
- Windows10 and Mac
- https://validator.w3.org was used to validate the HTML code in this project
- Linters used include Lighthouse, Webhint, and Stylelint
